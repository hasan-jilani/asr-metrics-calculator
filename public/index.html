Streaming TTS Alphanumeric Comparison

Compare ElevenLabs Flash v2.5, Cartesia Sonic-3, Deepgram Aura-2, and Rime Mist v2

How to Use This Demo

1
Choose a challenge: Select a pre-set challenge from the dropdown, enter custom text, or click "Generate Random Challenge" to create a new test case.
2
Select voices (optional): Choose different voices for each provider using the voice dropdowns. Changing a voice will reset that provider's audio.
3
Start the test: Click "Start TTS Test" to generate audio from all four providers simultaneously. The Time-to-First-Audio (TTFA) metric will be displayed for each provider.
4
Play & compare: Once buffering is complete, click "Play" buttons to hear each provider's pronunciation. Use "Download Audio" to save individual audio files.
